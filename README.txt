Эта программа используется для распознавания игральных карт из изображений определенного формата.
Программа не использует сторонние библиотеки по распознованию обьектов на изображениях. 
В основе реализованного мной алгоритма лежит метод нахождения расстояния Левенштейна.
Распознавание осуществляется по рангу карт и по масти.
Программа выводит в консоль информацию в формате <имя файла с изображением для распознавания> - <распознанные карты в мнемоническом виде>
Мнемонический вывод распознанных карт состоит из двухсимвольных наборов и включает в себя ранг карты(цифру или первую букву названия ранга на английском, 
за которой следует название масти карты на английском). Например, вывод 5cQs6h , означает 5 треф дама пик шестерка черви.
Для запуска программы запустите файл run.bat находящийся в корневой папке. 
Тестовые изображения находятся в папке image. 
Введите в командную строку "image", либо полный путь до другой папки с изображениями установленного образца (образец в папке image).
